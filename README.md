# Alexa Voice Assistant (Python)

A simple Python voice assistant that listens for the wake word "Alexa" and performs tasks using microphone input.

## Features
- Play songs
- Tell the current time
- Answer questions
- Tell jokes

## How to Run

Open terminal and run:
python alexa.py

Then say:
"Alexa play Shape of You"
"Alexa what time is it"
"Alexa who is Elon Musk"
